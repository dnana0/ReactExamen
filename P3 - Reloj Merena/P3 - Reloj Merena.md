# Notas P3 - Reloj Merena

## Descripción General

- Debes generar un reloj que le ayude a Shinnosuke a no perderse su serie de dibujos favorita: Ultrahéroe.
- Estos dibujos se emiten todos los días a las 17:00 horas (la hora de la "merena").
- El reloj debe mostrar las horas minutos y segundos que faltan desde la hora actual hasta el comienzo del show de dibujos animados.
- Si son las 17:00h o más tarde, el reloj debe mostrar 00:00:00.

## Calificación

- Seleccionar los elementos del DOM necesarios -> 10 puntos
- Generar un objeto Temporal API con la hora actual -> 15 puntos
- Generar un objeto Temporal API con la hora de la "merena" -> 15 puntos
- Realizar la aritmética/lógica de fechas necesaria -> 20 puntos
- Mostrar los resultados correctamente/actualizados -> 15 puntos
